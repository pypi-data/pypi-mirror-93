Detailed Description

This header file declares the functions required to compute the spherical harmonics decomposition for a particular sphere.

This header file declares the ProSHADE_internal_sphericalHarmonics namespace, which groups all the function required to compute the spherical harmonics decompostion on a surface of a sphere. There are also support functions for tasks like memory allocation and release and such.
