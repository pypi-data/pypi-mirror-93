
ipannotoryous
=====================================

Version: |release|

An annotation Jupyter Widget based on `Annotorius <https://github.com/recogito/annotorious>`_.


Quickstart
----------

To get started with ipannotoryous, install with pip::

    pip install ipannotoryous

or with conda::

    conda install ipannotoryous
